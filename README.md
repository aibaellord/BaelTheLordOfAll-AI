## рҹҶ• What's New in v2.1.0

### рҹӨ– True Autonomy

- **Auto-Setup Wizard**: One-command setup that discovers and configures everything
- **Service Discovery**: Automatically probes for LLM providers, MCP servers, databases
- **Self-Updating**: Discovers new models, integrations, and opportunities
- **MCP Client**: Not just a server - BAEL can now consume external MCP tools

### рҹҸӣпёҸ Real Council Deliberation

- **6 LLM-Backed Personas**: Sage, Guardian, Innovator, Analyst, Executor, Diplomat
- **Multi-Phase Deliberation**: Opinions вҶ’ Discussion вҶ’ Voting вҶ’ Decision
- **Parallel Processing**: Gather opinions simultaneously for speed
- **Deliberation History**: Track all council decisions

### рҹ–ҘпёҸ Advanced React UI

- **Dashboard**: Activity charts, system status, quick actions
- **Multi-Terminal**: xterm.js with split views and tabs
- **Chat Interface**: Streaming responses, message history
- **Council Visualization**: Watch deliberations unfold in real-time
- **Settings Pages**: LLM config, memory, integrations, security, appearance
- **Workflow Builder**: Visual node-based automation editor
- **Tools Explorer**: Browse, test, and execute tools

### рҹ”Ң Core Wiring Layer

- **LLMExecutor**: Real API calls to OpenRouter, Anthropic, OpenAI, Ollama
- **MemoryExecutor**: Connected working and long-term memory
- **UnifiedWiring**: Orchestrates all components together
