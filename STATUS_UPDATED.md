## Deployment Guide

### Quick Start

```bash
# One-command deployment (all phases)
python main.py --deploy docker-compose --env staging

# Or use orchestrator directly
from core.deployment import DeploymentOrchestrator, DeploymentTarget
orchestrator = DeploymentOrchestrator()
```
