# Deep Research Workflow

## Overview
This workflow enables comprehensive research on any topic using multi-source analysis.

## Steps

// turbo-all

1. **Define Research Scope**
   - Clarify the research question
   - Identify key concepts and terms
   - Set depth and breadth parameters

2. **Knowledge Base Search**
   ```bash
   grep_search for existing knowledge in /Volumes/SSD320/BaelTheLordOfAll-AI/
   ```

3. **Web Research**
   - Use search_web tool for current information
   - Use read_url_content for detailed analysis
   - Cross-reference multiple sources

4. **Code Analysis** (if applicable)
   - Use view_file_outline for structure
   - Use view_code_item for specifics
   - Trace dependencies and relationships

5. **Synthesis**
   - Combine findings from all sources
   - Identify patterns and insights
   - Generate actionable recommendations

6. **Documentation**
   - Create comprehensive report
   - Include citations and references
   - Store in knowledge base

## Output Format
Structured markdown with:
- Executive summary
- Key findings
- Detailed analysis
- Recommendations
- References
