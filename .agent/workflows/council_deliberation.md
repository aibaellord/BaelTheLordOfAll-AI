# Council Deliberation Workflow

## Overview
Simulate multi-persona council for complex decision making and consensus building.

## Steps

// turbo-all

1. **Problem Definition**
   - Clearly state the decision to be made
   - Define success criteria
   - Identify constraints

2. **Council Assembly**
   Convene specialized perspectives:
   - **Optimization Council**: Efficiency and performance
   - **Exploitation Council**: Resource maximization
   - **Innovation Council**: Novel approaches
   - **Validation Council**: Quality assurance
   - **Micro-Detail Council**: Edge cases

3. **Individual Analysis**
   Each council perspective analyzes:
   - Pros and cons from their viewpoint
   - Risks and mitigations
   - Implementation approach

4. **Deliberation**
   - Compare all perspectives
   - Identify conflicts and synergies
   - Seek consensus or weighted decision

5. **Decision Synthesis**
   - Combine best elements from each view
   - Create unified recommendation
   - Document dissenting opinions

6. **Implementation Planning**
   - Break decision into actionable steps
   - Assign priorities
   - Set validation criteria

7. **Execution**
   - Implement the decision
   - Monitor outcomes
   - Iterate if needed

## Council Weighting
- Critical decisions: All councils equal
- Performance focus: Optimization 2x
- Resource focus: Exploitation 2x
- New features: Innovation 2x
