## Usage Examples

### Deploy Multi-Team Agents

```python
from core.universal_agents import AgentTemplateLibrary, ProjectContext

library = AgentTemplateLibrary()
deployment = await library.deploy_for_project("vulnerability_hunter", project_context)
```

### Discover Opportunities

```python
from core.opportunity_discovery import OpportunityDiscoveryEngine

engine = await OpportunityDiscoveryEngine.create()
opportunities = await engine.discover_all("/path/to/project")
```

### Synthesize Reality

```python
from core.reality_synthesis import RealitySynthesisEngine, BranchingStrategy

engine = await RealitySynthesisEngine.create()
multiverse = await engine.synthesize(problem, BranchingStrategy.FIBONACCI)
```

### Dominate a Project

```python
from core.domination import AbsoluteDominationController, DominationMode

controller = await AbsoluteDominationController.create()
result = await controller.dominate(target, DominationMode.TRANSCENDENT, objectives)
```
