### Make a Request

```python
from api.api_gateway import get_api_gateway, Request, HTTPMethod

gateway = get_api_gateway()

    request_id="req_123",
    method=HTTPMethod.POST,
    path="/api/v1/reasoning",
    version="v1"
)
response = await gateway.handle_request(request)
```
